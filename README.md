🚗 Carnova
📱 Description

Carnova is an Android application that I developed using Kotlin and Firebase.
The idea behind this project was to create a simple and practical platform where users can explore different cars, check their details, and make a purchase.

✨ What the app does
Allows users to browse different types of cars
Displays detailed information about each car
Lets users buy cars directly from the app
Includes user authentication using Firebase
🛠️ Tech Stack
Kotlin
Firebase Authentication
Firebase Database
Android SDK
🔐 Authentication

User authentication is handled with Firebase Authentication.
Users can create an account, log in, and access the app securely.

🚀 How to run the project
Clone the repository:
git clone https://github.com/your-username/carnova.git
Open it with Android Studio
Add your Firebase configuration file (google-services.json)
Run the app on an emulator or a real device

This project was built as part of my learning journey in Android development.
It helped me understand how to connect a mobile app with a backend using Firebase and manage user authentication.
