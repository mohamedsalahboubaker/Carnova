# 🚗 Carnova

## 📱 Description
Carnova is an Android application that I developed using Kotlin and Firebase.  
The goal of this project was to build a simple and practical platform where users can explore different cars, view their details, and make a purchase.

---

## ✨ Features
- Browse different types of cars  
- View detailed information about each car  
- Purchase cars directly through the app  
- Secure user authentication using Firebase  

---

## 🛠️ Tech Stack
- Kotlin  
- Firebase Authentication  
- Firebase Database  
- Android SDK  

---

## 🔐 Authentication
User authentication is handled using Firebase Authentication.  
Users can create an account, log in, and securely access the application.
